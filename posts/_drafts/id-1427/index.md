I often have to deploy TE in a "clean slate" state for clients for testing purposes or to validate rules, content, etc but it's a pain to have setup a full console on Windows due to the lack of easy scripting (Linux, of course, is pretty easy in comparison!), so I figured I'd put together a quick script for this to make life a bit easier that might be helpful if you need to setup lab environments, etc. It's not perfect, as automation options for Windows is hit and miss (especially automating the Edge browser without plugins) - so effectively this script:

1. Automates the download of MS-SQL 2019 Developer Edition and installs it for you on the local C drive

3. Setups the TE database and user

5. Installs the latest version of Amazon's Java 11

7. Downloads TE from the TCC (it uses an automation of Microsoft Edge since the TCC is a bit "troublesome" for automation - so you need to be careful not to "interact" and interrupt the "playback" of the login and download process - you may also want to make sure you launch Edge first as by default now-a-days it does some super weird hit and miss UI behaviours (sometimes going full screen for a pointless first launch warning, sometimes giving toolbar related popups)! You will be prompted here to confirm the download (I could test for this continously, but I prefer a chance to check everything at this stage anyway!)

9. Unzips and runs the TE installer, setting a preconfigured passphrase set in the script

11. Watches the log file then runs through the initial post install in another Edge browser session (setting DB parameters, restarting the console and then setting the local administrator login) and does a final restart

13. Leaving you just to license (whilst this could be automated, since there's no "common" place to grab a license I've not automated this!) and get your set up going

Works fine for me on Windows Server 2019 (but _only_ if Edge is installed) and 2022 - you might want to poke firewall holes if you want to setup additional agents, but you'll hopefully spot that it's easy to adjust the script to, say, grab an older version of TE so if all you need is a single console this should hopefully save you some time!
