Recently I noticed a Tripwire feature looking for something similar to this so figured it would be a good topic to post here this week....

The script below generates reports including the Node name, IP, policy name and remediation information in a CSV format (ideal for ingestion by Archer GRC!) by merging multiple reports generated using TECommander. This does use some pretty old versions of my PowerShell calls, but it works just fine in 9.1/recent PowerShell versions so you should be good!

## Usage 

Running a report against a single node, with a single policy 

> ```
> DetailedRemediationReportV1.4.ps1 -TEusername administrator -TEpassword YourPasswordHere -FilterToNode "NODENAME" - FilterToPolicy "MS Windows Server 2012 R2 DM - CIS v2.2.1 Level 2" -output "D:\temp\outputfile.csv" 
> ```

Running a report against a node group (with one or more nodes in it), with a single policy 

> ```
> DetailedRemediationReportV1.4.ps1 -TEusername administrator -TEpassword YourPasswordHere -FilterToNodeGroup "Tripwire Enterprise Console" -FilterToPolicy "MS Windows Server 2012 R2 DM - CIS v2.2.1 Level 2" -output "D:\temp\outputfile.csv" 
> ```

Running a report against a node group (with one or more nodes in it), with a single policy, outputing to another location (C:\\temp)

> ```
> DetailedRemediationReportV1.4.ps1 -TEusername administrator -TEpassword YourPasswordHere -FilterToNodeGroup "Tripwire Enterprise Console" -FilterToPolicy "MS Windows Server 2012 R2 DM - CIS v2.2.1 Level 2" -output "C:\temp\output.csv"
> ```
