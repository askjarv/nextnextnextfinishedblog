Scanning with IP360 "on demand" is a common request (for those with both TE and IP360 it's a handy "action" to have for example, or if you're reconciling against discovered assets in, say a cloud environment, this is an option for discovery) so having a way to trigger a scan from the CLI/via script is handy... 

So the following is just a quick example of doing an adhoc scan using Python 3 (most of my previous efforts and online examples tend to be Python 2, and I wanted to add this so I've got an easy note to remind me that an "adhoc" scan requires a post against AUDITS and not "Scan Profiles" which I always forget when I come back to this after working with other products for a while!). 

NB: This is not necessarily something you'll use "as is" since you'll likely need to add logic to, say, find the network you wish to scan, etc. but it is a useful one that I often share with clients when they ask about how to get started with IP360's API and they're not super confident with the IP360's API docs/python _but_ it is simple enough that they get a scan going in a few minutes and feel "productive":

```
#-*- coding: utf-8 -*-"""@author: Tripwire PS""""""DEPENDENCIES:"""""" Required to do all HTTP request calls to the API """import requestsimport urllib3""" Required to confirm server hostname exists and is resolvable and validate IP's"""import socket,struct""" You'll probably want to parse some JSON as some stage!"""import json"""GLOBAL VARIABLES:"""# These are the static parameters needed for the connection to the IP360 VnEVNE="YourVNEhere"BASEURL="https://"+BASEURL+"/rest/v1"USERNAME="ip360@tripwire.com"PASS="YourPasswordHere"# NetworkID should an id for a network which includes the intended scan target IP - you can check for the network ID on the Network widget in IP360NETWORKID=2RANGETOSCAN="192.168.0.72/32"# You can get a scan profile ID from your VnE's Scan Profile widgetSCANPROFILEID="5""""FUNCTIONS:"""# Functions to validate IP addresses in rangesdef makeMask(n): "return a mask of n bits as a long integer" return (2<<n-1) - 1def dottedQuadToNum(ip): "convert decimal dotted quad string to long integer" return struct.unpack('L',socket.inet_aton(ip))[0]def networkMask(ip,bits): "Convert a network address to a long integer" return dottedQuadToNum(ip) & makeMask(bits)def addressInNetwork(ip,net): "Is an address in a network" return ip & net == net"""BEGIN CODE:"""# Set up Session() to not show warnings/errors for the self-signed certsession = requests.Session()session.verify = Falseurllib3.disable_warnings()session.headers.update({ "accept": "application/json, text/plain", "X-Requested-With": "required"})# Get networksnetworks = session.get(url="%s/networks" % (BASEURL), auth=(USERNAME, PASS))networks.connection.close()# Get Scan Profilesprofiles = session.get(url="%s/scan_profiles" % (BASEURL), auth=(USERNAME, PASS))profiles.connection.close()# Get Scan Configsconfigs = session.get(url="%s/scan_configs" % (BASEURL), auth=(USERNAME, PASS))configs.connection.close()# Post an Adhoc Audit Scan (assumes you've only got one appliance pool to use)PAYLOAD={ "debug": False, "name":"Test7", "network":"https://"+VNE+"/rest/v1/networks/"+NETWORKID, "pool":"https://"+VNE+"/rest/v1/appliance_pools/1", "range":RANGETOSCAN, "scan_profile" : "https://"+VNE+"/rest/v1/scan_profiles/"+PROFILEID, "scan_profile_type": "Discovery", "scan_type": "Partial", "special_scan": False}postscan = session.post(url="%s/audits" % (BASEURL), auth=(USERNAME, PASS),json=PAYLOAD)postscan.connection.close()
```
