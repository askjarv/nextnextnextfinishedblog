Trend Worry-Free Business Security 7 - Messaging Agent

2011-03-03

If like me you didn’t know that the Messaging Agent (ScanMail) in Trend Worry-Free Business Security 7 uses SQL Express 2005 - Then you might want to know that the 64-bit SQLEXPR.exe used doesn’t work!

I’ve worked out that you need to download the correct file from Microsoft (http://go.microsoft.com/fwlink/?linkid=83387) and replace the file located in \\\ofcscan\MSA\x64\SQL2005Express
